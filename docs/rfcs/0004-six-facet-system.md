# RFC 0004: The NOUS Six-Facet System

Status: accepted

Date: 2026-07-18

## Decision

NOUS is represented as a six-facet system, rather than a linear history of
successive product categories:

1. space;
2. exhibition;
3. network;
4. data curation;
5. knowledge base;
6. intelligent agent.

These facets can be active together. A NOUS experiment, space, exhibition,
project, or agent is a situated expression of the six facets with a specific
distribution of emphasis.

## Vocabulary

`knowledge base` is the system-level term because it names the durable,
source-aware, navigable body of materials and relations.

`knowledge environment` remains useful for describing an interactive runtime
or interface that operates on a knowledge base. It is not the name of the
fifth system facet.

`intelligent agent` is the Chinese-facing system term. `Creative Agent` remains
the name of the WEME implementation direction.

## Case Rule

The Vault is a case of the six-facet system, not a public identity parallel to
NOUS. Its primary facets are knowledge base and intelligent agent; data
curation is a supporting facet. Its dedicated URL remains a runtime endpoint
under `/the-vault/`.
