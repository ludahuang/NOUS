策展孵化器 / Curator Incubator

NOUS 从问题出发，组织空间、展览、网络、数据、知识与智能体，让策展在共同参与、持续记录和再次激活中生成。

05 / KNOWLEDGE BASE 知识库

将材料、笔记、来源与关系保存为可检索、可导航、可再次激活的共同记忆。

HOW NOUS OPERATES

NOUS 让关系持续生成

NOUS 从一个问题开始，建立可进入的界面，邀请不同主体共同参与，并把发生过程转化为可以继续工作的记录与关系。

NOUS begins with a question, builds an interface for participation, and carries what happens forward as traceable relationships that can be activated again.

05 / KNOWLEDGE BASE
组织可继续工作的共同记忆

将分散材料组织为可检索、可导航的共同记忆，支持持续研究与再次激活。

策展让关系持续生成

每一次 NOUS 发生，都从问题开始，经过临时界面、共同参与和结构化记录， 最终在新的地点、媒介或智能体中再次激活。

  1. 01 提出问题

    确定值得共同进入的未知，而不是预设一个封闭答案。

  2. 02 建立界面

    空间、展览、网站、知识图谱或智能体都可以成为临时界面。

  3. 03 邀请主体

    让艺术家、研究者、公众、组织与机器进入同一关系场。

  4. 04 生成与记录

    保存作品，也保存来源、条件、关系、工具和未解决的问题。

  5. 05 再次激活

    把记录转化为可传送、可修订、可被下一种形态继续使用的结构。
