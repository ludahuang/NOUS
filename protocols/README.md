# Protocols

Protocols describe how NOUS operates across locations, media, projects, and
agents.

Initial protocol families:

- `curation/` - how questions, participants, works, and relations become a program;
- `replication/` - how a form can be transferred and rebuilt;
- `provenance/` - how claims, sources, transformations, and rights are recorded;
- `publishing/` - how public releases remain attributable and reversible;
- `agent/` - how a creative agent receives authority and leaves an audit trail.
