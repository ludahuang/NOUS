# 预测加工与主动推断

预测加工把感知理解为由先验模型与感觉误差共同约束的生成过程；主动推断进一步把行动视为改变输入和维持生命状态的方式。

## 核心命题

- 知觉不是被动读取，而是受证据约束的模型更新。
- 注意可以被理解为对预测误差精度的调节。
- 身体调节和行动为预测模型提供持续校准。
