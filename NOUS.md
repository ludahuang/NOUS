# NOUS

## 关于 NOUS

NOUS 在法语中是第一人称复数“我们”，在英语思想传统中指向智性、
知性与心智，也可以被理解为一种共同感知和常识。

NOUS 的名称最早出现在微米公司的主体信息和官方邮件前缀中。它后来
成为由鲁大荒发起的实验空间、集装箱系统、策展机制、跨城市连接方式、
知识环境，并继续演化为 Creative Agent。

这些形态不是彼此替代的版本，而是同一机制在不同媒介中的具体化：

```text
空间 -> 展览 -> 网络 -> 数据策展 -> 知识环境 -> Creative Agent
```

## NOUS 做什么

NOUS 关注：

- 关于展览的展览；
- 关于创造的创造；
- 关于服务的服务；
- social innovation 与 rapid prototyping；
- 在商业条件下催生非商业主体；
- 在艺术土壤中生成非艺术空间；
- 将策展从地点组织转化为可复制、可传送、可协作的协议。

## 空间原型

NOUS 空间以 40 英尺 ISO 集装箱作为重要原型。标准化结构使空间能够：

- 被运输和快速组装；
- 在不同城市与尺度中重新出现；
- 通过图纸、数据与远程协作完成本地制造；
- 承载实时影像、声音、展览与公共活动；
- 形成开放链式的策展和内容传送机制。

因此，集装箱不是最终作品，而是连接艺术、技术、城市与公共空间的
可移动接口。

## 策展机制

NOUS 的基本策展循环是：

```text
提出问题
  -> 建立临时空间或界面
  -> 邀请不同主体进入
  -> 生成作品、关系和记录
  -> 将结果结构化为可传送的数据
  -> 在新的地点、媒介或智能体中再次激活
```

策展的对象不仅是作品，也包括关系、条件、工具、记忆、来源和未来的
再生成能力。

## Creative Agent

Creative Agent 不是给 NOUS 增加一个聊天窗口，而是让策展机制获得一个
可持续行动的主体。

一个 NOUS Creative Agent 应当：

- 有明确身份与审美立场；
- 能访问经过授权的资源；
- 能提出关系、缺口与未来问题；
- 能协助生成作品、研究和策展方案；
- 对来源、工具调用和生成过程留下记录；
- 不在没有授权的情况下替人发布、删除或作出不可逆决定；
- 能被评估、修订、迁移和归档。

## 创始人与策展人

鲁大荒，1972 年生，现居北京，跨媒介艺术家、微米公司创始人、
NOUS 策展人。完成法国巴黎八大 Hypermedia 学业后，他持续发起媒体、
艺术、教育、社会创新和技术项目，并在创作者、企业家、写作者、
策展人、设计师、工程师与开发者之间工作。

## English Summary

NOUS is not a single venue or application. It is a curatorial mechanism that
has moved through container architecture, exhibitions, networked media, data
curation, knowledge environments, and creative agents.

Its enduring principles are mobility, replicability, open participation,
source awareness, distributed production, and the ability to reactivate an
idea in a new place or medium.
