# Licensing and Rights

The repository currently contains a historical root `LICENSE` with GPL v3
terms. That single file is not sufficient for a mixed cultural repository.

NOUS needs separate decisions for:

- source code;
- written content and translations;
- photographs, video, sound, and design assets;
- catalog metadata and datasets;
- generated material;
- model weights or agent configurations;
- third-party source material.

Until a right is confirmed, catalog records use `NOASSERTION` and historical
sites are treated as reference-only sources.

No existing license is changed by this architecture scaffold.
