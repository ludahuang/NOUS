# The Vault Resources

Resources are source materials used to generate or verify curated vaults.

They remain distinct from generated vault output:

```text
source-materials/   editorial source documents
../vaults/          generated Obsidian-compatible vaults
../scripts/         deterministic generators and validators
```

Do not represent generated notes as independent historical sources. Preserve
the source-material path in note frontmatter and the generated vault manifest.

Rights for imported source material remain `NOASSERTION` until explicitly
documented.
