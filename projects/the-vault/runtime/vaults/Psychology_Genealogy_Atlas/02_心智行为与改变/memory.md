# 记忆

记忆不是固定档案的回放，而是在当前线索、身体状态和既有模型约束下进行的重构过程。

## 核心命题

- 编码、巩固与提取由不同机制和时间尺度支持。
- 记忆连接学习、预测、情绪和自传身份。
- 遗忘既可能是限制，也可能是适应性选择。
