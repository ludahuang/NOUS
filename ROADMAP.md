# NOUS Roadmap

Updated: 2026-07-23

## Phase 1 - Establish the Mother Repository

- define the NOUS identity and root information architecture;
- create the entity catalog and schema;
- establish provenance, archive, publishing, and agent protocols;
- inventory the historical sites without copying unlicensed media;
- document the migration path for The Vault.

Exit condition:

- a reader and a machine can identify the main NOUS entities and relationships;
- no active product needs to occupy the repository root.

## Phase 2 - Recover and Describe the Historical Network

- describe the container spaces, exhibitions, people, works, and cities;
- map stable identifiers to historical URLs;
- record source dates, revisions, rights, and media availability;
- identify missing materials and preservation risks;
- create preservation packages for selected born-digital works.

## Phase 4 - Formalize WEME Creative Agent

- define identity, aesthetic position, authority, and memory policy;
- register tools, resources, prompts, and workflows;
- add provenance for model and tool activity;
- establish evaluation fixtures for curatorial and creative behavior;
- connect approved NOUS catalog resources.

- build a public map across spaces, exhibitions, projects, agents, and sources;
- expose provenance and preservation status;
- support bilingual reading;
- publish project-specific entry points without collapsing them into one product;
- offer machine-readable catalog exports.
