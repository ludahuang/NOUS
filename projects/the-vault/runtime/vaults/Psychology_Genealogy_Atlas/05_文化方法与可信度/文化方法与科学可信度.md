# 文化方法与科学可信度

心理学知识由样本、测量工具、研究制度和文化前提共同生产；可信度来自持续暴露假设、纠正误差和比较不同主体位置。

## 核心命题

- 研究对象和正常标准具有历史与文化条件。
- 测量、实验和统计模型都需要构念效度。
- 开放科学把心理学理解为持续纠错的程序而非固定答案库。

- 标准化提高比较性，也可能抹平文化差异。
- 批判分析需要提出可行动的替代研究设计。
