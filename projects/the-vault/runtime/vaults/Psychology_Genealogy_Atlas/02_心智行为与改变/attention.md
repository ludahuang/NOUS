# 注意

注意调节信息的优先级和精度，但它与意识并不等同：信息可以受到注意而未必成为体验，也可能被体验而未被主动报告。

## 核心命题

- 注意受到目标、显著性、习惯和身体需求共同影响。
- 选择性注意可以增强部分信息并抑制竞争信息。
- 注意既是认知控制机制，也是意识理论中的关键争议点。
