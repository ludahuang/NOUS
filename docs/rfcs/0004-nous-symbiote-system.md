# RFC 0004: The NOUS Symbiote System

Status: accepted

Date: 2026-07-18

## Decision

NOUS is represented as a symbiotic curatorial organism, rather than a linear
history of successive product categories:

1. space;
2. experience;
3. network;
4. data curation;
5. knowledge base;
6. intelligent agent.

These functions can be active together. The intelligent agent is the central
acting subject. Space, experience, network, data curation, and knowledge base
surround and sustain it. A NOUS experiment, space, exhibition, project, or
agent is a situated expression of these six functions with a specific
distribution of emphasis.

The public Atlas represents the system as a 2.5D meteor-like structure with an
almond-kernel silhouette and a sharp, continuous outer boundary. The exterior
does not use a beveled rim. A central pentagon represents the intelligent
agent. Five surrounding facets represent space, experience, network, data
curation, and knowledge base. Visitors can drag the organism and activate the
center or any surrounding facet; the selected function updates its mechanism
explanation.

The irregular form is semantic. Project-specific emphasis expands or contracts
the surrounding contour and the prominence of the central agent. Every NOUS
project or case therefore produces its own symbiotic form.

## Vocabulary

`knowledge base` is the system-level term because it names the durable,
source-aware, navigable body of materials and relations.

`experience` names the second system function. Historical exhibitions remain
cataloged as exhibitions; the vocabulary change does not rewrite those records.

`knowledge environment` remains useful for describing an interactive runtime
or interface that operates on a knowledge base. It is not the name of the
fifth system function.

`intelligent agent` is the Chinese-facing system term. `Creative Agent` remains
the name of the WEME implementation direction.

## Case Rule

The Vault is a NOUS Symbiote weighted toward a prominent central intelligent
agent and a strong knowledge-base outer facet, not a public identity parallel
to NOUS. These two functions remain the stable primary definition of the case
while any individual facet is being inspected. Data curation is a supporting
function. Its dedicated URL remains a runtime endpoint under `/the-vault/`.
