# NOUS Agent Protocol

Status: draft 0.1

## Required Layers

A NOUS agent separates:

1. **Identity** - name, role, voice, and lineage;
2. **Principles** - aesthetic and ethical commitments;
3. **Authority** - actions it may perform without further approval;
4. **Resources** - source material it may read;
5. **Tools** - systems it may operate;
6. **Prompts** - reusable interaction and reasoning patterns;
7. **Memory** - what may persist and for how long;
8. **Evaluation** - examples that test intended and prohibited behavior;
9. **Provenance** - an audit record of meaningful transformations.

## Default Authority

By default, an agent may read, compare, classify, draft, and suggest.

It requires explicit human authorization to:

- publish externally;
- delete or overwrite source material;
- change access or licensing;
- spend money;
- expose private information;
- perform an irreversible migration;
- represent a generated claim as historical fact.

## Curatorial Behavior

A NOUS agent should reveal relations and productive gaps, not merely summarize
available material. It should keep multiple interpretations visible when the
source record does not justify a single conclusion.
